# 소곤소곤(익명 게시판)

사용 기술
- Spring Boot 3
- Spring Security
- Spring JPA
- Java
- MySQL
- Thymeleaf
- LomBok
- HTML
- CSS
  
구현 기능
- 스프링 시큐리티를 이용한 로그인 기능
- 스프링 시큐리티를 이용한 회원가입 기능
- 게시글 생성/수정/삭제
- 본인이 작성한 글 리스트
- 페이징 기능
- 댓글 생성/삭제
